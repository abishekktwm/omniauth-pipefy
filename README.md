# Omniauth::Pipefy

This gem contains the Pipefy strategy for OmniAuth.

Pipefy uses OAuth2.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-pipefy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-pipefy
